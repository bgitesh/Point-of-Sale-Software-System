# **Open Source Point of Sale (OSPOS)**  

A web-based point-of-sale system built using PHP, MySQL, and CodeIgniter 4, featuring stock management, taxation, invoicing, barcode generation, multi-user support, reporting, and more.

## **Key Features:**
- Inventory, sales, and expenses management  
- Multi-language support, user permissions, and GDPR compliance  
- Reporting tools and integrations (MailChimp, Google reCAPTCHA)  
- **Demo available**: [Live Demo](https://demo.opensourcepos.org/)  
  - Username: `admin`, Password: `pointofsale`

---

## **Installation:**  
- Follow instructions in [INSTALL.md](INSTALL.md).
